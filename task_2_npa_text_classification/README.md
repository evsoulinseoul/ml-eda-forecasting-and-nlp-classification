# Задание 2. Классификация нормативно-правовых актов

В задании решается задача мультиклассовой классификации нормативно-правовых актов по тексту документа. В качестве признаков используются TF-IDF-представления словных и символьных n-грамм, а в качестве моделей — линейные классификаторы.

Дополнительно выполнены анализ иерархии целевой переменной и проверка TF-IDF-признаков на возможные утечки: даты, номера документов, годы, технические коды и административные маркеры.

## Файлы

- `task2_npa_tfidf_classification.ipynb` — основной ноутбук с кодом, метриками и выводами.
- `task2_npa_tfidf_classification.py` — версия решения в формате Python-скрипта.
- `data/README.md` — инструкция по размещению исходного parquet-файла.
- `outputs/` — папка, куда сохраняются таблицы метрик, отчёт классификации, примеры ошибок и итоговая модель.

## Данные

Исходный файл для второго задания не включён в репозиторий из-за большого размера. Перед запуском положите его в папку:

```text
task_2_npa_text_classification/data/
```

и переименуйте в:

```text
internship_50k.parquet
```

## Запуск

Откройте ноутбук `task2_npa_tfidf_classification.ipynb` в Jupyter Notebook или VS Code и выполните все ячейки сверху вниз.

Основные результаты сохраняются в папку:

```text
outputs/
```
