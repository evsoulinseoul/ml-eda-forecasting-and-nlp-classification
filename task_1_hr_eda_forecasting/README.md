# Задание 1. HR-аналитика и прогноз нагрузки на рекрутинг

В задании выполнен разведочный анализ данных по IT-направлениям и построен прогноз общей нагрузки на рекрутинг до 2027 года.

## Файлы

- `task1_hr_eda_forecast.ipynb` — основной ноутбук с кодом, графиками и выводами.
- `data/df_ITs.csv` — исходный датасет для задания 1.
- `outputs/` — папка, куда сохраняется итоговый CSV с прогнозом.

## Запуск

Откройте ноутбук `task1_hr_eda_forecast.ipynb` в Jupyter Notebook или VS Code и выполните все ячейки сверху вниз.

Итоговый файл с прогнозом сохраняется в:

```text
outputs/forecast_2025_2027_task1.csv
```
