## Задание 1. HR-аналитика: EDA и прогнозирование

В первом задании выполнен разведочный анализ данных по IT-направлениям. Рассмотрены основные показатели нагрузки на рекрутинг: количество обработанных резюме, входящие заявки, найм, увольнения, текучесть, размер команды, активные проекты, зарплатный бюджет и другие признаки.

После анализа динамики показателей построен прогноз общей нагрузки на рекрутинг до 2027 года. Результат сохраняется в CSV-файл в папку `outputs/` внутри задания 1.

## Задание 2. Классификация нормативно-правовых актов

Во втором задании решается задача мультиклассовой классификации нормативно-правовых актов по тексту документа. Используется связка TF-IDF + линейные модели. В решении дополнительно рассмотрена иерархия целевой переменной и выполнена проверка признаков на возможные утечки.

По результатам экспериментов итоговой моделью выбрана линейная SVM-модель на объединённых word и char TF-IDF-признаках. В ноутбуке сохранены метрики, анализ ошибок и выводы по влиянию технических маркеров в тексте.

Датасет для второго задания не включён в репозиторий из-за большого размера. Для запуска нужно положить файл `internship_50k.parquet` в папку `task_2_npa_text_classification/data/`.

## Установка зависимостей

Рекомендуется создать виртуальное окружение:

```bash
python -m venv .venv
```

Активация на Windows:

```bash
.venv\Scripts\activate
```

Установка библиотек:

```bash
pip install -r requirements.txt
```

## Запуск

Для выполнения заданий откройте нужный ноутбук в Jupyter Notebook или VS Code и последовательно выполните все ячейки сверху вниз.

```text
task_1_hr_eda_forecasting/task1_hr_eda_forecast.ipynb
task_2_npa_text_classification/task2_npa_tfidf_classification.ipynb
```

Файлы результатов создаются в папках `outputs/` соответствующих заданий.
